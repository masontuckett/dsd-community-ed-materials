# dsd-community-ed-materials

Independently developed courses for [Davis School District's Community Education program](https://daviscomed.davis.k12.ut.us/)—designed for both adult and youth learners!

## COURSES
- Intro to Cybersecurity
- Computer Networking
- Ethical Hacking
- Intro to Linux
- Python Programming
- Web Development

## COURSE INFO

### Intro to Cybersecurity
__A practical overview of foundational cybersecurity concepts:__ explore how computers work, how
networks function, and how hackers operate. In this course we’ll practice using real tools,
familiarize ourselves with computer networking, improve our online security, and enhance our
digital privacy!

### Computer Networking
__A beginner-friendly primer on computer networking:__ learn how the internet functions,
troubleshoot common internet related issues, familiarize yourself with common networking
protocols, learn how to configure IP addresses and VLANs, gain a working understanding of
DHCP/DNS, and general security best practices!

### Ethical Hacking
__A beginner-friendly, practical introduction to ethical hacking and penetration testing:__ explore
open-source intelligence, crack passwords, map computer networks, and find weaknesses! In
supervised labs we’ll practice safe reconnaissance, basic scanning, simple web security, and
compete in mini capture-the-flag challenges using legally approved targets!

### Intro to Linux
__A beginner-friendly, practical introduction to Linux:__ learn how the operating system works,
explore the desktop, navigate the command line, manage files and permissions, install software,
and troubleshoot networking! In this course we’ll be practicing our skills with Linux Mint!

### Python Programming
__A beginner-friendly, practical introduction to Python programming (learn Python fundamentals):__
syntax, variables, operators, conditional statements, functions, lists, dictionaries, exception
handling, and working with files. In this course we familiarize ourselves with general
programming and develop a working understanding of Python!

### Web Development
__A beginner-friendly, practical introduction to developing a portfolio site:__ learn basic HTML/CSS,
explore potential frameworks, and establish a brand around your projects. In this course we will
learn how the World Wide Web functions, project development, and hosting options!​
